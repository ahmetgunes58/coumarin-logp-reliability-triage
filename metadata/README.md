# Metadata

This folder contains repository-level integrity and citation metadata.

## Files

SHA256SUMS.txt

SHA256SUMS.txt lists SHA256 hashes for the final repository files. It can be used to verify that files were not modified during transfer, download, or archival deposition.

The checksum file is regenerated after any root-document or data update.
